# Features

NELA combines multiple local workflows in one desktop application:

- **Local Indexing**: Build a private knowledge base from your project files.
- **Private Inference**: Run chat, vision, and generation tasks on-device.
- **Model Management**: Download, switch, and tune models in Settings.
- **UI Integration**: Use workspace-centric navigation with mode-based interactions.
